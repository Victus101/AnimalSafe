# AnimalSafe Business Rules

AnimalSafe helps users report animals in danger and coordinate rescue.

## User Roles

### USER

Can:

- Register
- Login
- Create reports
- View reports

### VOLUNTEER

Can:

- View reports
- Update report status
- Participate in rescue workflows

### ADMIN

Can:

- Manage users
- Delete reports
- Access admin endpoints

## Report Lifecycle

OPEN → IN_PROGRESS → RESOLVED

## Rescue Logic

1. User reports animal
2. Volunteer accepts case
3. Rescue happens
4. Report marked resolved

## Main Objective

Reduce abandoned or endangered animals through coordinated reporting.
