# 🐾 ¿Qué es AnimalSafe?

AnimalSafe es una plataforma mobile-first enfocada en ayudar animales heridos, abandonados o en peligro mediante colaboración comunitaria.

La aplicación permite a personas reportar casos cercanos, visualizar animales necesitados en un mapa y facilitar que voluntarios o rescatistas actúen rápidamente.
