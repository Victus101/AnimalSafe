🧠 Reglas UX Importantes
Siempre preguntar:
¿el usuario entiende esto en 2 segundos?
¿la acción principal está clara?
¿hay demasiados pasos?
¿se puede usar rápido en calle?
🚫 Errores a Evitar
pantallas cargadas
texto pequeño
botones escondidos
demasiados colores
formularios largos
listas antes que mapa
🎯 KPI UX MVP

Usuario debe poder:

abrir app
loguearse
ver mapa
reportar animal
cerrar en menos de 2 min
